HTML, CSS, and JavaScript

This article provides a complete overview of HTML, CSS, and JavaScript for building interactive and visually appealing web pages.

HTML Basics

HTML provides structure using elements like headings, paragraphs, and links.

<!DOCTYPE html>
<html>
  <head><title>My Page</title></head>
  <body>
    <h1>Welcome</h1>
  </body>
</html>

How to Add CSS

  • Inline: Using the style attribute inside HTML tags.
  • Internal: Using a <style> tag in the <head>.
  • External: Linking a separate CSS file using <link>.

HTML Headings

HTML has six levels of headings: <h1> to <h6>.

HTML Class Attribute

Used to apply styles via CSS or select with JS.

<div class="highlight">Styled Text</div>

HTML Colors

Use color names, HEX codes, or RGB values.

HTML Background Color

Example using inline style:

<body style="background-color: lightblue;">

Primer CSS

Primer CSS is GitHub’s CSS framework.

Difference Between HTML and CSS

  • HTML – structure
  • CSS – styling

CSS History and Versions

Introduced in 1996. Key versions: CSS1, CSS2, CSS3. Features like Flexbox and Grid are modern additions.

CSS Selectors

Used to target HTML elements.

  • p – element selector
  • .myClass – class selector
  • #myId – ID selector
p { color: white; }
.myClass { background-color: yellow; }
#myId { font-size: 20px; }

CSS Box Model

Includes content, padding, border, and margin.

div {
  padding: 10px;
  border: 2px solid green;
  margin: 20px;
}

CSS Flexbox

Helps in creating responsive layouts.

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

CSS Media Queries

Adapt your design for different screen sizes.

@media (max-width: 768px) {
  body { font-size: 14px; }
}

CSS Animations

Used to animate transitions.

@keyframes slide {
  from { left: 0; }
  to { left: 100px; }
}
div { animation: slide 2s infinite; }

JavaScript Introduction

JavaScript adds interactivity and logic to web pages.

JavaScript Basics

Example using variables and functions:

let name = "John";
function greet() {
  console.log("Hello, " + name);
}
greet();

JavaScript and the DOM

Access and modify HTML using JavaScript:

document.getElementById("myElement").textContent = "Updated!";

JavaScript Example

<button onclick="changeText()">Click Me</button>
<p id="demo">Original Text</p>
<script>
function changeText() {
  document.getElementById("demo").textContent = "Text Changed!";
}
</script>
